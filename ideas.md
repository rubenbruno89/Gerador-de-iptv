# Ideias de Design — Gerador de URL HLS & TS

<idea>
**Design Movement:** Cyberpunk Terminal / Dark Tech
**Core Principles:**
- Interface escura com acentos em verde neon ou ciano, evocando terminais de programação
- Tipografia monospace para os campos de URL gerados, reforçando o aspecto técnico
- Bordas finas e brilhantes com efeito de "glow" nos elementos interativos
- Layout centralizado mas com assimetria sutil nos detalhes decorativos

**Color Philosophy:** Fundo quase preto (#0d1117), texto principal em cinza claro (#c9d1d9), acentos em verde neon (#39ff14) ou ciano (#00e5ff). Transmite poder técnico e precisão.

**Layout Paradigm:** Card único centralizado com fundo escuro e partículas/grid decorativo. Campos de input com bordas neon.

**Signature Elements:**
- Bordas com efeito glow nos inputs e botões
- Ícones de terminal/código ao lado dos campos

**Interaction Philosophy:** Animações de digitação ao gerar URLs; botão de copiar com feedback visual neon.

**Animation:** Fade-in suave no card; pulse no botão de gerar; texto das URLs aparece como se estivesse sendo "digitado".

**Typography System:** JetBrains Mono para URLs e labels técnicas; Inter para textos de suporte.
</idea>
